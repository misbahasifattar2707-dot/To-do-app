Todo App

Part 5: Todo CRUD

What's new in this part?
  • CREATE - Add new todos (POST /api/todos)
  • READ - Get all todos (GET /api/todos)
  • UPDATE - Edit/complete todos (PUT /api/todos/:id)
  • DELETE - Remove todos (DELETE /api/todos/:id)
Security Note: In this part, there's no authentication on todo routes. Anyone could access anyone's todos by changing user_id! We'll fix this in Part 6.
