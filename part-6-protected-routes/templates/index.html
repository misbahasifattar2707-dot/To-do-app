Todo App

Part 6: Protected Routes

What's new in this part?
  • @token_required decorator - Protects API routes
  • Authorization header - "Bearer token" sent with requests
  • Ownership verification - Users can only access their own todos
How @token_required works:
# Without protection (Part 5):
@app.route('/api/todos')
def get_todos():
    user_id = request.args.get('user_id')  # Anyone can change this!
    ...

# With protection (Part 6):
@app.route('/api/todos')
@token_required
def get_todos(current_user):  # User comes from token!
    todos = Todo.query.filter_by(user_id=current_user.id).all()
    ...
